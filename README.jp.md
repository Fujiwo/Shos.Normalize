# Shos.Normalize

Shos.Normalize は、クリップボード内のテキストを正規化する Windows 向けツールです。異なるアプリケーションからコピーしたテキストに互換文字や引用符が含まれている場合に、より一貫した形式へ変換できます。

## 主な機能

- Unicode の NFKC 正規化として `NormalizationForm.FormKC` を適用します。
- 円記号 (`￥`) を ASCII のバックスラッシュ (`\`) に変換します。
- 左右の二重引用符 (`“` と `”`) を ASCII の二重引用符 (`\"`) に変換します。
- 右シングル引用符 (`’`) を ASCII のアポストロフィ (`'`) に変換します。
- 半角濁点 (`ﾞ`) を全角濁点 (`゛`) に変換します。
- 半角半濁点 (`ﾟ`) を全角半濁点 (`゜`) に変換します。
- 対応するテキスト形式のみで構成されるクリップボードのデータを処理します。ロケール情報の併存は許可します。
- 正規化結果が元のテキストと異なる場合にだけ、クリップボードへ書き戻します。

## 動作環境

- Windows
- .NET 10

## 使い方

1. テキストをクリップボードにコピーします。
2. `Shos.Normalize.exe` を実行します。
3. 正規化されたテキストを必要な場所へ貼り付けます。

対応するテキスト形式が存在しない場合、テキストが空の場合、または HTML・リッチテキスト・画像・ファイル一覧などの非対応形式が一つでも含まれる場合、アプリケーションはクリップボードを変更せずに終了します。処理中に例外が発生した場合は、エラーメッセージをコンソールへ出力します。

## ビルド方法

### Visual Studio

Visual Studio で `Shos.Normalize.slnx` を開き、通常の「ビルド」コマンドでソリューションをビルドします。

### .NET CLI

リポジトリのルートで次のコマンドを実行します。

```powershell
dotnet build Shos.Normalize.slnx
```

プロジェクトのターゲットフレームワークは `net10.0-windows` で、クリップボードへのアクセスに Windows Forms を使用しています。
